Project Title

Drug Analyser for Patients Using Chatbots

Abstract

The Drug Analyser for Patients Using Chatbots is an AI-based healthcare solution that allows patients to receive real-time drug-related information through natural language interaction. The system, built using Flutter and integrated with Gemini API, delivers personalized responses about medication dosage, side effects, interactions, and reminders. This SEPM project focused on sprint-based development, agile execution, and real-world problem-solving aligned with SDG 3: Good Health and Well-being.

Learning Outcomes

  • Agile sprint execution and user story writing
  • Project planning with backlog and release tracking
  • Hands-on testing, debugging, and UI design
  • API integration using Gemini for AI-powered responses
  • Team collaboration and daily stand-ups

Sprint Summary

Sprint Focus Outcome
Sprint 1 Initial Setup & Core Features Chatbot framework and Gemini API integrated
Sprint 2 UI and Error Handling Loading spinner, modular code, prompt parsing
Sprint 3 Performance & Theme UI theming, validations, error dialogs
Sprint 4 Testing & Documentation Test cases, screenshots, responsive UI
Sprint 5 Deployment & Final Report APK, walkthrough, user guide, final documentation

Tools & Technologies
